# PowerBall Lottery Generator - Web Version

A web-based PowerBall number generator that uses historical drawing data to generate ticket suggestions.

## Features

- Generate tickets based on historical frequency analysis
- Hot/Cold number ratings
- Add new drawings (saved to browser localStorage)
- Upload custom data files
- No server required - runs entirely in browser

## Deployment

This is a static site that can be hosted on any web server or static hosting service.

### Files
- `index.html` - Main entry point
- `src/` - JavaScript files
- `styles/` - CSS files

### Coolify Deployment
1. Connect your GitHub repository
2. Set build pack to "Static"
3. Set publish directory to `/` (root)
4. Deploy

## Disclaimer

This tool is for entertainment purposes only. Lottery drawings are random and past frequency does not predict future results.
