# Product Context

## Why This Project Exists
A learning project to build a functional desktop application using Electron while exploring frequency analysis and weighted random selection algorithms.

## Target User
- Single user (developer)
- Technical but still learning
- Uses Cursor AI to assist development

## User Experience Goals
1. Simple, single-window interface
2. One-click ticket generation
3. Option to generate multiple tickets at once
4. Clear display of generated numbers
5. Informative statistics panel

## What It Does
- Analyzes historical PowerBall drawings to find frequency trends
- Generates lottery ticket suggestions weighted by historical frequency
- Tracks session statistics (tickets generated, coverage percentage)

## What It Does NOT Do
- Predict winning numbers
- Claim increased odds or advantage
- Use machine learning or AI models
- Persist data between sessions
- Auto-fetch data from lottery websites
